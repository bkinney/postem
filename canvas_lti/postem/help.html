Glossary

Click on a term to reveal its definition

  • Grades
    • The existing Canvas Gradebook. If Grades meets your needs, you should not install Post'Em for Canvas
  • CSV
    • A comma-separated values file. This is the simplest filetype supported by applications such as MS Excel. Although the word 'comma' appears in the name, other characters, such as a tab or semi-colon may be used to separate columns, while new lines separate rows. Your Post'Em upload files MUST be comma separated. For best results, download the template provided by Post'Em, edit it, and re-upload.
  • Unpublished Files
    • The Canvas Files tool is used to store your Post'Em data. Post'em files are unpublished, so that students can not open them outside the context of the Post'Em tool. This prevents students from viewing one another's grades.
  • Student View
    • When a student visits the Post'Em tool, he will see his own grades, listed with one row per assignment. There are two ways to preview Post'Em as a student. Using Post'em's 'View as student', or switching to Canvas's student view. The latter will only show grades if you have awarded some to the Canvas Test Student.
  • Instructor View
    • After uploading a csv file to Post'Em, you will see a full table containing all the data. Use the show/hide editor link if you want to hide one or more assignment columns from students. Students will never see the SSID column.
  • External Tool (aka App)
    • An application hosted outside of Canvas that conforms to the IMS LTI standard for interoperability. External Tools are installed by visiting the Apps tab within course Settings.
  • Logging out
    • Logging out of a Post'Em session is a good idea, especially if you have installed any other UD hosted External Tool, such as Peer Evaluation, Export Rubric Scores, or Critboard.

Overview

Post'Em for Canvas@UD is a replacement gradebook for those who prefer to store and calculate grades on their personal computer. It is very similar to the same-named Sakai tool.

How to Use this Page

The glossary on the left should help you understand what the features of the tool are, and how these are intended to be used. Step-by-Step instructions for setting up and using Post'Em as well as an FAQ can be located in the accordion panels below. Click on a panel's title to expand it.

Step-by-Step

  • Installing Post'Em
    1. Launch Canvas, and browse to the course in which you wish to use Post'Em
    2. Click the Settings link located at the bottom of the Canvas navigation menu, located at the left side of the screen.
    3. Click on the Navigation tab.
    4. Locate the Post'Em item, located toward the bottom of the page. Click and drag this item into the list of active navigation links.
    5. Scroll to the bottom of the page and click 'Save'. It is a good idea to perform this step only when you have some data to upload.
  • Download your template
    1. Click on the Post'Em link, and click the 'Download Template' link. A file will be downloaded to your computer. The exact behavior will depend upon your choice of browser, and the preferences you have set.
    2. Save the file in the folder of your choice on your hard drive.
  • Edit your file
    1. Open the template file using Excel, or other spreadsheet software. Add assignment columns as needed. If necessary, you can delete or add rows, being very certain to get copy student SSIDs correctly.
    2. Enter grade data. These can be either numbers or text.
    3. Save your changes. DO NOT change the filetype from .csv to any other type.
  • Upload your file
    1. If you have just finished editing your file, be sure to CLOSE IT. Excel often locks files, making them impossible to upload.
    2. Open the Post'em tool and click 'Choose File'. Browse to the location where your .csv file is stored.
    3. Wait for your filename to appear beside the 'Choose File' button. The Submit button should now be active.
    4. Click 'Submit'.

FAQ

Click on a question to reveal its answer

  • Why can't I use the Post'Em editor to hide the SISID column?
    • Both the Name and SISID columns hidden in student view. Only assignment columns can be edited directly within the Post'Em tool. Students will see their names at the top of the page (replacing the title text, 'Instructor View').
  • What should I do if one or more students drop or add the class?
    • It is best to wait until after drop/add is over before downloading your course template, however, it is possible to add or remove rows from your csv prior to uploading it.
  • What is the best way to edit an uploaded file?
    • Do not attempt to edit a Post'em file after upload. Instead, edit the original csv file, and re-upload. The new file will replace the existing one.
